What is differences between HTML,CSS,and JavaScrip?

Write down your own answer in 3 paragraph

HTML :HTML (Hypertext Markup Language) is used to structure the content of a webpage,
defining elements like headings, paragraphs, lists, links, images, and more. It forms the foundation of a webpage's structure.

JavaScrip :JavaScript is a programming language that adds interactivity to web pages. It allows you to create dynamic content, manipulate the webpage's behavior, and interact with users.
With JavaScript, you can respond to user actions, validate forms, and update content without needing to reload the entire page.

css :CSS (Cascading Style Sheets) is used to control the presentation and layout of web content.
It defines how HTML elements should be styled, specifying attributes like colors, fonts, spacing, and positioning. CSS separates the visual design from the content itself.
